#火焰大挑戰

### 需求 
* `盈泰工業社 地轉車 100-B-000066201 100-B-000097160`轉成`盈泰工業社 地轉車 000066201 000097160`
* 每個年度分別歸在一個json檔
* 最後完成目標是在line機器人輸入 100-B-000066202 會跑出 盈泰工業社 地轉車
* 可把這些json檔案放在網路上再使用http呼叫
* `100-B-000097160`英文為檢驗機構代碼，需要區分
	* 100 為年號
	* B為檢驗機構代碼
	* 000097160為號碼
